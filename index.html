Dynamic Countdown Timer

Please provide a target time via a URL parameter to start the countdown.

How to use:

Add ?time= to the end of the URL, followed by a date in ISO 8601 format.

Example (UTC time):
?time=2025-12-25T10:00:00Z

Example (With timezone offset):
?time=2026-01-20T15:30:00-05:00
